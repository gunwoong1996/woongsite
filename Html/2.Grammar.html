h1~h6

숫자가 커질수록 글자크기가 작아지며 강조 및 줄바꿈이 기본으로 적용된다.

줄바꿈

  • br(braek) - 단순 줄바꿈으로 닫는 태그가 존재하지않는다. c의 \n과 동일

  • p(paragraph) - 문단을 나타내며 줄바꿈시 일정 공백란이 생긴다.

  • pre - 태그 내에 있는 엔터와 스페이스가 그대로 웹 브라우저에 적용되어 보여지게 된다.

    중요한 부분 강조

    Strong - 화면낭독기 반응(웹브라우저에서 중요한부분)

    B - 단순 시각적

    U - 밑줄(Underline의 약자)

    에셋 삽입

    img - 사진 삽입

    iframe - 비디오 삽입

    박건웅,권준석,오선재 졸업작품 (평행세계)

    목록

    ul(unorderd list) - 리스트 묶음(list의 부모태그)

    ol(orderd list) - 리스트 묶음 숫자내림차순(list의 부모태그)

    list - 리스트 생성

    1. 일번
    2. 이번
    3. 삼번
    • 철수

    • 길동

    표

    table - 테이블 전체묶음

    thead - 테이블 헤드

    th - 셀을 테이블 셀그룹의 헤더로 지정(글씨 진해짐)

    tb(table body) - 테이블 바디

    tr(table row) - 테이블 행(가로)

    td(table data) - 테이블 내용

    tfoot - 항상 가장 아래행에 표시

    border - 테이블 격자(테이블태그 내에서 사용)

    rowspan - 열병합

    colspan - 행병합
